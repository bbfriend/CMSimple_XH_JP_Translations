Twocents_XH – a CMSimple_XH Plugin

Twocents_XHは訪問者の意見・考え(Two cents)を追加することができます。あなたは、複数のページの一般的なコメントとしてそれを使用するか、または単一ページ上のゲストブック、または両方。 Realblog_XH　のコメントプラグインとして使用することができます。必要に応じてHTMLのシンプルな WYSIWYGエディタで入力することができる、利用可能です。

オプションでは、新しいコメントが投稿された時に通知すること。コメントを自動的に公開または承認制（確認後に公開）コメントの管理を有効にすること。、conforming CAPTCHA pluginを使用して、スパム防止を追加することができます.

Twocents_XH is merely a draft—many features are missing yet, and what is there might change. It is a so called alpha version. Use with utmost caution!

Requirements

Twocents_XH is a plugin for CMSimple_XH. It requires CMSimple_XH ≥ 1.6 and PHP ≥ 5.1.

インストール

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory twocents/ to your server into CMSimple_XH's plugins/ directory.
  4. Set write permissions for the subdirectories css/, config/, languages/ and htmlpurifier/standalone/HTMLPurifier/DefinitionCache/Serializer/.

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select "Twocents" from "Plugins".

Localization is done under "Language". You can translate the character strings to your own language (if there is no appropriate language file available), or customize them according to your needs.

The look of Twocents_XH can be customized under "Stylesheet".

Usage

To place a comment facility on a page, write:

{{{twocents('%TOPICNAME%');}}}

%TOPICNAME% can be an arbitrary name, but it may contain only the letters a-z, the digits 0-9 and minus signs (-). The topicname is not necessarily related to the page heading; in fact it is just the name of a file that is stored in the subfolder twocents/ of CMSimple_XH's content folder. If you like you can display the comments for the same topic on different pages.

Some examples:

{{{twocents('guestbook');}}}
{{{twocents('article-1');}}}

Note that there must be at most one call of twocents on a single page.

Administration

管理者としてログインしているとき、編集/表示/非表示や各コメントを削除するためのボタンが表示されます。

In the plugin administration area you can convert existing comments from resp. to HTML, depending on the setting of the configuration option Comments → Markup. Note that it's best to do this only when the site is in maintenance mode, and that you have to change this configuration option after the conversion.

Furthermore there is the possibility to import comments from the Comments plugin. To use this, you have to copy the data files of the Comments plugin into Twocents_XH's data folder. Note that some of the information will be ignored, e.g. the IP address and the uploaded image, as well as the markup when Twocents_XH is in plain text mode.

Limitations

Currently, posting comments from very old browsers such as IE 6 and IE 7 is not possible; the form submission will silently fail.

Twocents_XH uses HTML5 form validation features. These will be critized by validators for HTML 4.01 and XHTML 1.0 templates. Other than that they cause no harm; old browsers will simply ignore them, and contemporary browsers will heed them nonetheless.
