Fotorama_XH - a CMSimple_XH Plugin

Fotorama_XH は、 Fotorama ギャラリーを CMSimple_XH に移植したものです. Fotorama_XHは、画像のアップロード機能は持っていません。 CMSimple_XH'のimages フォルダー内に事前にアップロードされた JPEG画像のみを表示します。 Every gallery can have its own settings, and every imagecan have an additional caption.

インストール

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory fotorama/ to your server into CMSimple_XH's plugins/ directory.
  4. Set write permissions for the subdirectories cache/, css/, config/ and languages/.
back to top

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select "Fotorama" from "Plugins".

You can change the default settings of Fotorama_XH in "Config". Hints for the options will be displayed when hovering over the help icon with your mouse.

Localization is done in "Language". You can translate the character strings to your own language, if there is no appropriate language file available, or customize them according to your needs.

The look of Fotorama_XH can be customized in "Stylesheet".

Usage

Prepare a gallery

  1. imageのサブフォルダに画像をアップロード
    CSimple_XHのfilebrowser（ファイル管理 > イメージ）やFTPソフトでimage以下にサブフォルダを作成し、画像をアップロードします。
  2. ギャラリー定義XMLファイルを作成
    プラグイン管理　＞　Fotorama 　＞ギャラリー　新規ギャラリー名と、そのギャラリーの対象とするフォルダを選択、作成します。
    新規ギャラリー名は、半角小文字(a-z)､半角英数字(0-9)、ハイフン(-)を使用します。
    　content/fotorama/ギャラリー名.xml で保存されます。
  3. XMLファイルを編集（必要に応じて）
    サンプル：
    <gallery width="600" ratio="4/3" nav="thumbs" fullscreen="true" transition="crossfade" path="対象とするフォルダ">
      1. width and ration
        These attributes specify the width and aspect ratio of the gallery,respectively. The width is either a plain number giving the width in pixels(e.g. "400") or a percentage of the available horizontal space (e.g. "100%"),which is especially useful for responsive layouts. The ratio is either a fraction (e.g. 400/300 or 16/9) or a decimal numer (e.g. 1.3333). If these attributes are omitted, the width and aspect ratio are determined by the first image. Note that the images will be resized to fit within the width/ratio, so that it's possible to have portrait and landscape images mixed in the samegallery without distortion.
      2. nav
        Only "thumbs" is allowed if the attribute is specified. This will turn the slim "dot" navigation into a thumbnail navigation. The required thumbnails are automatically generated on the fly, and stored in the plugin's cache directory.
      3. fullscreen
        This allows the visitor to enter fullscreen mode. Choose either "true", what will restrict the fullscreen mode to the browser window, but also works for older browsers, or "native" what uses the full screen size if supported by the browser.
      4. transition
        Either "slide" (the default if the attribute is omitted), "crossfade" or "dissolve". The latter is probably only useful if you have images that differ only slightly; otherwise "crossfade" is preferable.

After having successfully created the XML file, you are redirected to the gallery editor where you can fine-tune the gallery by editing the XML file. You can remove and add <pic> elements, and reorder them. For every <pic> element you can optionally specify a caption attribute whose value will be shown in the gallery; the caption will also be used as alt attribute of the HTML <img>. You can change the value of the path attribute, but you must not remove the attribute completely. Note that you should not touch the first three lines of the file (the XML and the doctype declaration).

Embed a gallery

To embed a gallery on a page simply write:

{{{fotorama('%NAME%');}}}

where %NAME% is the name of the gallery, e.g.

{{{fotorama('holidays');}}}

Limitations

For the galleries to be fully functional, JavaScript has to be enabled in the visitor's browser.
