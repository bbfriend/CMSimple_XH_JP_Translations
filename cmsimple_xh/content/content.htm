重要なアドバイス

テキストエディターで config.php　に書かれてるパスワードは変更しないでください。
デフォルトのパスワードは、ログインしてからオンラインで書き換えてください。

CMSimple_XHファイルを編集するには、"UTF-8N : BOMなしのutf-8"としてファイルを検出、オープン、保存できる エディタ
 Windows: TeraPad  秀丸  サクラエディタ など
 MacOS : CotEditor  KEdit など
を使用してください。

IMPORTANT ADVICE

DO NOT change the password directly in config.php by using a text editor. Change the default password right away after login ONLINE (login with default password).

This is the easiest and best working way.

For editing CMSimple_XH files it is essential to USE ONLY editors, capable of detecting, opening and saving files as "utf-8 without BOM" (Byte Order Mark), like notepad++.

If this condition is not fulfilled, you will damage your CMSimple installation.

WICHTIGER HINWEIS

Bitte ändern Sie das Standard Passwort NICHT direkt in der config.php mit einem Text Editor. Ändern Sie das Standard Passwort nach dem Login ONLINE  (Login mit dem Standard Passwort).

Zur Bearbeitung der CMSimple_XH Systemdateien sollten Sie ausschliesslich einen Code Editor wie z. B. notepad++ benutzen, der die Codierung "utf-8 ohne BOM" (Byte Order Mark) erkennt, die Dateien entsprechend öffnet und auch wieder abspeichert.

Wenn Sie die Systemdateien in einer anderen Codierung als "utf-8 ohne BOM" abspeichern, kann es zu Problemen mit verschiedenen Funktionen von CMSimple_XH kommen.

Menu Levels & Headings

 CMSimple_XHは、見出し(HTMLのh1~h6タグ)に、特殊な意味を持ちます。
　それは、CMSimple_XHは、

  1. 各ページのコンテンツを　１枚のデータファイルとして保存 し、
  2. 保存された1枚のファイルをHタグ（デフォルトでh1~h3)で分割し､各ページとして表示

　する為です

一般に、見出し(h1~h6タグ)とは、書籍で例えれば、各章の題目、章の名称です。HTMLには、大きなものから小さなものまで、6つの見出し、h1〜h6 があります。

CMSimple_XHは、

  • 大きい方の見出し＝分割用　：データファイルの分割に使用　し、
  • 小さい方の見出し＝段落用　：各ページ内の見出し:段落 として使用　

します。書籍では、各章内で小さい見出しを　段落の見出し として 使用されます。

大きな見出しは、設定/CMS設定 の　Menu Levels　で設定します。デフォルトは3で、この場合、
h1-h3タグ は

 

  • データファイルの分割用に使用される
  • 各ページのトップに表示される項目名、ページ名、タイトルに使用される
  • メニュー（目次）に使用される　
    残りの小さい見出しh4、h5、h6は各ページ内の 段落 として使用され、メニューに表示されません。 

　　　*　編集モードで　各ページの　Metaタグ や ページ設定　で代替は可能

 

Menu Levelsに4 を設定すると、新しいWebページの分割に h4タグ が使用されます。 Menu Levelsに2を設定すると、h3とh4を1枚のWebページ内の段落 として使用できます。 しかし、ほとんどの場合、3つのレベルが良い妥協です。

メニューレベルのデフォルトは、3 （h1〜h3はページの分割に使用される）。
各ページ内の段落には見出しh4-h6 を使用する。

CMSimple_XH 1.5から、プラグインPagemanagerが含まれています。 Pagemanagerは、ページ管理を提供し、Webページの作成、名前変更、移動、削除を簡素化します。
